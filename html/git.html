Git

  • git clone - клонирование удаленного депозитория на локальный компьютер.
  • git status - просмотр изменений в папке.
  • git commit -m - создание коммита, показывающего, что файл изменен.
  • git push - внесение изменений на удаленный депозиторий.
  • git log - выводит список всех коммитов.
